Observable

A simple library for keeping up to date with an object when it changes.

Learn more

Simple, unobtrusive data synchronization

  • Set up complex object dependencies once, then sit back and watch as they're automatically kept in sync.
  • Create functions that update when their underlying Observables change.
  • Based on simple JavaScript types. String, Array, Function, Number, and Boolean types are all supported.

Installation

npm install o_0
bower install o_0#dist-v0.3.0
